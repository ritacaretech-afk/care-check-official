# Care Check 見やすい記録表示版

## 変更内容
食事・水分・服薬、排泄の記録を入力したあと、上部に大きな確認カードとして表示される仕様に変更しました。

## 改善点
- 入力後の内容が一目で確認できる
- 食事・水分・服薬は、朝昼夕ごとの内容と1日水分合計をカード表示
- 排泄は、最終排尿・前回排便日・最新記録・排便換算をカード表示
- 申し送り、管理者確認、記録漏れ確認がしやすい画面構成

## 使い方
1. 今の care-check/src をバックアップ
2. このZIPを解凍
3. src_visible_records を src に名前変更
4. care-check フォルダへ入れる
5. VS Codeで npm run build
6. 成功したら git add . / git commit / git push
